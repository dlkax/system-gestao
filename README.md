# system-gestao
Desenvolvimento de um Sistema de Gestão para Pequenos Negócios Locais

 PROBLEMA

Pequenos negócios como lava-rápidos, oficinas, salões e comércios locais ainda utilizam papel, WhatsApp ou planilhas desorganizadas para controlar clientes, serviços, estoque e faturamento, o que gera erros, retrabalho e perda de oportunidades.

 OBJETIVO GERAL

Desenvolver um sistema de gestão simples, acessível e eficiente para pequenos negócios, visando melhorar a organização administrativa, o controle financeiro e a produtividade operacional.

 OBJETIVOS ESPECÍFICOS

Criar um sistema de cadastro de clientes, serviços e produtos
Implementar controle de atendimentos/agendamentos
Desenvolver relatórios básicos de faturamento
Garantir autenticação e segurança dos dados
Oferecer uma interface simples e intuitiva

 JUSTIFICATIVA

Este projeto contribui socialmente ao apoiar pequenos empreendedores que não possuem acesso a sistemas profissionais, além de proporcionar ao estudante experiência prática em levantamento de requisitos, arquitetura de software, banco de dados, backend, frontend e testes, alinhando teoria e prática na Engenharia de Software.

 PÚBLICO-ALVO

Microempreendedores e pequenos negócios locais, como lava-rápidos, salões de beleza, oficinas mecânicas e pequenos comércios.

 FUNCIONALIDADES DO SISTEMA

Login e controle de usuários
Cadastro de clientes
Cadastro de serviços/produtos
Registro de atendimentos
Relatórios financeiros simples
Painel administrativo

 TECNOLOGIAS:

Backend: Python (Flask ou FastAPI)
Banco de dados: SQLite ou PostgreSQL
Frontend: HTML, CSS, JavaScript
Autenticação: JWT ou sessão
Hospedagem: Render / Railway / VPS

 METODOLOGIA

Levantamento de requisitos com pequenos empreendedores
Modelagem do sistema (casos de uso, DER, fluxogramas)
Desenvolvimento incremental
Testes funcionais

Implantação piloto em negócio real

Avaliação dos resultados
